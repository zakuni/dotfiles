dotfiles of zakuni, by zakuni, for zakuni
===

## Configurations Included
 - emacs
 - fish
 - tmux
 - vim
 - zsh

## Prerequisites
 - [Cask](https://github.com/cask/cask)

## Using vimrc 
    % ln -s dotfiles/.vimrc ~/.vimrc
    % ln -s dotfiles/.vimrc ~/.gvimrc
    % ln -s dotfiles/.vim ~/.vim
then open Vim and`:BundleInstall`
